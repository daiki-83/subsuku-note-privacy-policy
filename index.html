01

はじめに

「サブスクノート」（以下「本アプリ」）をご利用いただきありがとうございます。本アプリの開発者（以下「当方」）は、ユーザーのプライバシーを尊重し、個人情報の適切な取り扱いに努めています。

本プライバシーポリシーは、本アプリにおける情報の収集・利用・管理方針についてご説明するものです。

02

収集する情報について

当方は、本アプリを通じてユーザーの氏名・メールアドレス・電話番号などの個人を直接特定できる情報を収集することはありません。

ただし、以下のサードパーティサービスを通じて、技術的な情報が自動的に収集される場合があります。

  • 広告の配信最適化・パフォーマンス計測のための広告識別子（広告 ID / IDFA）
  • アプリの利用状況・クラッシュレポートなどの診断情報
  • デバイスの種別・OSバージョン・言語設定などの技術情報
03

広告について（バージョン 2.0 以降）

Google AdMob

本アプリはバージョン 2.0 より、Google LLC が提供する広告配信サービス「Google AdMob」を導入しています。AdMob は、広告識別子（Android 広告 ID）を使用して、ユーザーの興味・関心に基づいたパーソナライズ広告を表示する場合があります。

  • 収集データ：広告 ID、おおよその位置情報（国・地域レベル）、アプリの利用状況、デバイス情報
  • 利用目的：広告の表示・最適化、不正防止、利用統計の集計

Google のプライバシーポリシー：
https://policies.google.com/privacy

広告 ID のリセットやパーソナライズ広告のオプトアウトは、デバイスの設定から行えます。

  • Android：設定 → Google → 広告 → 「広告 ID を削除する」または「広告のパーソナライズを無効にする」
04

アナリティクスについて

Firebase Analytics

本アプリは、Google LLC が提供する「Firebase Analytics（Google Analytics for Firebase）」を使用してアプリの利用状況を分析しています。収集されたデータは、機能改善やユーザー体験の向上を目的として利用されます。

  • 収集データ：起動回数、画面遷移、操作イベント、クラッシュ情報、デバイス情報
  • 利用目的：バグ修正・機能改善・UX 最適化

Firebase のプライバシーポリシー：
https://firebase.google.com/support/privacy

05

データの第三者提供について

当方は、以下の場合を除き、収集したデータを第三者に販売・提供・開示することはありません。

  • ユーザーの同意がある場合
  • 法令に基づき開示が求められる場合
  • 上記 03・04 項に記載のサードパーティサービス（Google LLC）によるデータ処理
06

データの保存・管理

本アプリに入力されたサブスクリプション情報（サービス名・金額・更新日など）は、ユーザーのデバイス内にのみ保存されます。当方のサーバーへはアップロードされません。

07

子どものプライバシーについて

本アプリは 13 歳未満の子どもを対象としておらず、意図的に 13 歳未満の子どもから個人情報を収集することはありません。万一、13 歳未満の子どもに関する情報が収集されていると判明した場合は、速やかに削除対応を行います。

08

プライバシーポリシーの変更について

当方は、法令の改正やサービス内容の変更に伴い、本プライバシーポリシーを予告なく変更する場合があります。変更後のポリシーは本ページに掲載した時点から効力を生じます。定期的にご確認いただくことをお勧めします。
